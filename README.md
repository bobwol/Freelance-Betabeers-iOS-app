Instalación
-----------
* git clone git://github.com/miquelcamps/Freelance-Betabeers-iOS-app.git
* git submodule init
* git submodule update

Metodo de emergencia
--------------------
si quieres volver a la ultima versión, borra el contenido de la carpeta del repo y ejecuta
git reset --hard HEAD
